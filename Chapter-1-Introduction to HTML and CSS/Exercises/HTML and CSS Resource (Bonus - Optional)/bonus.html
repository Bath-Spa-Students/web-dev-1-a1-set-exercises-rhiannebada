HTML & CSS QUICK REFERENCE GUIDE

HTML TAGS

HTML tags are like keywords which defines that how web browser will format and display the content. With the help of tags, a web browser can distinguish between an HTML content and a simple content. HTML tags contain three main parts: opening tag, content and closing tag. But some HTML tags are unclosed tags.

When a web browser reads an HTML document, browser reads it from top to bottom and left to right. HTML tags are used to create HTML documents and render their properties. Each HTML tags have different properties.

  • <!--...-->: This tag defines a comment. Comment serves as guides for programmers, it does not show on final outputs.

    Example:

    <p> Example paragraph. </p> <!-- This is a comment -->


  • <a>: This tag defines a hyperlink, which is used to link from one page to another.

    Example:

    <a href = "https://bathspa.ac.ae/"> Bath Spa University Academic Centre RAK Website </a>

    output:

    Bath Spa University Academic Centre RAK Website 


  • <b>: This tag is used to make a text bold.

    Example:

    Normal Text, </b>Bold Text</b>

    output:

    Normal Text, Bold Text


  • <br>: It stands for breakline, it breaks the line of the code.

    Example:

    <p> Rhianne Ghernelle Intal Bada </p>

    <br>

    <p> Creative Computing <p>

    <p> Level 4 <p>

    output:

    Rhianne Ghernelle Intal Bada


    Creative Computing

    Level 4 


  • <h1>to <h6>: It defines headings for an HTML document from level 1 to level 6.

    Example:

    <h1>Heading 1</h1>

    <h2>Heading 2</h2>

    <h3>Heading 3</h3>

    <h4>Heading 4</h4>

    <h5>Heading 5</h5>

    <h6>Heading 6</h6>

    output:

    Heading 1

    Heading 2

    Heading 3

    Heading 4

    Heading 5
    Heading 6
  • <i>: Defines a part of text in an alternate voice or mood. It displays text in italics.

    Example:

    My name is <i>Rhianne Ghernelle Intal Bada</i>

    output:

    My name is Rhianne Ghernelle Intal Bada

  • <p>: It represents a paragraph in an HTML document.

    Example:

    <p>This is a paragraph.</p>

    output:

    This is a paragraph.

  • <small>: Defines smaller text. It is used to make text font one size smaller than document's default font size.

    Example:

    <small>Small text</small>

    output:

    Small Text

  • <sub>: It defines a text which displays as a subscript text.

    Example:

    Text <sub>Subscript</sub>

    output:

    Text Subscript

  • <sup>: It defines a text which represent as superscript text.

    Example:

    Text <sup>Superscript</sup>

    output:

    Text Superscript

CSS PROPERTIES

CSS (Cascading Style Sheets) is a language used to describe a document's presentation and formatting in HTML or XML. It allows web developers to control the appearance of elements on a web page.

  • align-content: Specifies the alignment between the lines inside a flexible container when the items do not use all available space

    Example:

    align-content: center;

  • background-color: Specifies the background color of an element

    Example:

    background-color: #ffffff;

  • border-bottom: A shorthand property for border-bottom-width, border-bottom-style and border-bottom-color

    Example:

    border-bottom: 1px solid #000000;

  • box-shadow: Attaches one or more shadows to an element

    Example:

    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);

  • color: Sets the color of text

    Example:

    color: #ff0000;

  • font-family: Specifies the font family for text

    Example:

    font-family: 'Montserrat', sans-serif;

  • margin: Sets all the margin properties in one declaration

    Example:

    margin: 10px;

  • padding: A shorthand property for all the padding-* properties

    Example:

    padding: 10px;

  • text-align: Specifies the horizontal alignment of text

    Example:

    text-align: center;

  • width: Sets the width of an element

    Example:

    width: 100px;
